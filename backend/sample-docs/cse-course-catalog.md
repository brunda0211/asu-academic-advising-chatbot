## Lower Division

### CSE 110 — Principles of Programming
**Credits:** 3  
**Prerequisites:** MAT 170 or MAT 171 (may be taken concurrently)  
**Description:** Introduces the fundamental concepts of computer programming using Java. Topics include variables, data types, control structures, methods, arrays, and basic object-oriented programming.

### CSE 205 — Object-Oriented Programming and Data Structures
**Credits:** 3  
**Prerequisites:** CSE 110 with a grade of C or better; MAT 170 with a grade of C or better  
**Description:** Covers object-oriented programming concepts including inheritance, polymorphism, interfaces, and abstract classes. Introduces fundamental data structures: linked lists, stacks, queues, and binary trees. Includes algorithm analysis basics.

### CSE 230 — Computer Organization and Assembly Language
**Credits:** 3  
**Prerequisites:** CSE 110 with a grade of C or better  
**Description:** Introduction to computer organization, instruction set architecture, assembly language programming, and basic digital logic. Topics include data representation, CPU architecture, memory hierarchy, and I/O systems.

## Upper Division

### CSE 310 — Data Structures and Algorithms
**Credits:** 3  
**Prerequisites:** CSE 205 with a grade of C or better; MAT 243 with a grade of C or better  
**Description:** Advanced data structures including balanced trees, hash tables, heaps, and graphs. Algorithm design techniques: divide-and-conquer, dynamic programming, greedy algorithms. Complexity analysis and NP-completeness introduction.

### CSE 330 — Operating Systems
**Credits:** 3  
**Prerequisites:** CSE 230 with a grade of C or better; CSE 310 with a grade of C or better  
**Description:** Fundamental concepts of operating systems including process management, memory management, file systems, and I/O. Topics include scheduling algorithms, synchronization, deadlock, virtual memory, and system calls.

### CSE 340 — Principles of Programming Languages
**Credits:** 3  
**Prerequisites:** CSE 310 with a grade of C or better  
**Description:** Formal language theory, syntax and semantics of programming languages, parsing techniques, type systems, scope and binding, functional programming, and logic programming paradigms.

### CSE 355 — Introduction to Theoretical Computer Science
**Credits:** 3  
**Prerequisites:** CSE 310 with a grade of C or better; MAT 243 with a grade of C or better  
**Description:** Formal models of computation including finite automata, pushdown automata, and Turing machines. Regular and context-free languages. Decidability and computational complexity theory.

### CSE 360 — Introduction to Software Engineering
**Credits:** 3  
**Prerequisites:** CSE 205 with a grade of C or better  
**Description:** Software development lifecycle, requirements engineering, software design patterns, testing methodologies, version control, agile development practices, and team collaboration tools.

### CSE 412 — Database Management
**Credits:** 3  
**Prerequisites:** CSE 310 with a grade of C or better  
**Description:** Relational database design, SQL, normalization theory, transaction processing, query optimization, and introduction to NoSQL databases. Includes a semester-long database project.

### CSE 434 — Computer Networks
**Credits:** 3  
**Prerequisites:** CSE 330 with a grade of C or better  
**Description:** Network architecture and protocols, TCP/IP stack, routing algorithms, network security fundamentals, wireless networks, and distributed systems concepts.

### CSE 450 — Design and Analysis of Algorithms
**Credits:** 3  
**Prerequisites:** CSE 310 with a grade of C or better; CSE 355 with a grade of C or better  
**Description:** Advanced algorithm design and analysis. Topics include network flow, linear programming, approximation algorithms, randomized algorithms, and amortized analysis.

### CSE 461 — Introduction to Artificial Intelligence
**Credits:** 3  
**Prerequisites:** CSE 310 with a grade of C or better  
**Description:** Foundations of AI including search algorithms, knowledge representation, machine learning basics, neural networks, natural language processing, and computer vision fundamentals.

### CSE 471 — Introduction to Machine Learning
**Credits:** 3  
**Prerequisites:** CSE 310 with a grade of C or better; MAT 243; STP 281 or equivalent statistics course  
**Description:** Supervised and unsupervised learning, regression, classification, clustering, dimensionality reduction, ensemble methods, and deep learning fundamentals. Hands-on projects using Python and scikit-learn.

## Graduate Courses

### CSE 511 — Data Processing at Scale
**Credits:** 3  
**Prerequisites:** Admission to CS graduate program or instructor approval  
**Description:** Large-scale data processing systems, distributed computing frameworks (Spark, Hadoop), stream processing, data warehousing, and cloud-based data architectures.

### CSE 575 — Statistical Machine Learning
**Credits:** 3  
**Prerequisites:** CSE 471 or equivalent; linear algebra proficiency  
**Description:** Advanced machine learning theory including kernel methods, graphical models, variational inference, reinforcement learning, and generative adversarial networks.
